Component Features Test

Mouse Controls:

  • Click and drag to move both images
  • Shift + click and drag to move only one image
  • Mouse wheel to zoom in/out
  • Ctrl/Cmd + hover for magnifier (8x zoom)

Keyboard Shortcuts:

  • v - Simple vertical wipe
  • w - Full wipe interface
  • a - Show only image A
  • b - Show only image B
  • u - Under composite mode
  • o - OnionSkin composite mode
  • d - Diff composite mode
  • i - InvDiff composite mode
  • c - Toggle checkerboard background
  • r - Reset view
  • h or ? - Toggle help screen

Wipe Interface (Full Mode):

  • Center dot - drag to move wipe line
  • Outer dot - drag to rotate wipe line
  • Arc handle - drag to adjust alpha blending

File Loading:

  • Click A or B boxes to upload files
  • Drag and drop images anywhere
  • Drop single image anywhere (except B box) to load as A
  • Drop multiple images to load first as A, second as B
